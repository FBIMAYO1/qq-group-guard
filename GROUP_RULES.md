# 狗三の群规 🐧

> 本群由狗三（凑企鹅）AI 辅助管理，违规行为会被自动检测。以下为群规细则。

---

## 一、核心原则

本群倡导**文明交流、友善互损、适度摸鱼**。违规判定以**语境和意图**为准，不是抓关键词。

狗三用 AI 语义分析判断违规，能识别谐音、拼音缩写、拆字、符号替换等一切变体。别以为换个写法就检测不到。

---

## 二、违规类别（7 类，零容忍）

### 🤬 1. 辱骂 / 人身攻击

**禁止**：恶意辱骂、诅咒家人、人格侮辱、群体攻击。

包括但不限于：傻逼(nmsl/sb)、你妈死了、草泥马(cnm)、杂种、去死、全家死光、你算什么东西 等。

**注意**：朋友间的玩笑互损（"你真狗""笨蛋""傻瓜"）不算违规。狗三看的是**恶意程度**，不是关键词。

---

### 🔞 2. R18 / 色情内容

**禁止**：约炮(yp)、性暗示、色情资源分享、裸聊(ll)、嫖娼(pc)、低俗擦边球。

包括但不限于：月抛、瑟琴(sq)、做爱(za)、开车暗示、"有资源""发福利""私聊看""包夜" 等。

**注意**：正常的恋爱话题、两性科普讨论不违规。

---

### 🩸 3. 暴力血腥

**禁止**：宣扬暴力、虐待动物/人、自残教学、血腥图片视频、砍人/杀人威胁。

---

### 💀 4. 违法内容

**禁止**：毒品(dp/xd)、诈骗(zp)、假证、代考、枪支、违禁品买卖。

包括但不限于：冰毒(bd)、大麻(dm)、假证(加正)、代考(带考) 等。

---

### 🚫 5. 政治敏感

**禁止**：分裂主义言论、邪教宣传、敏感历史事件、侮辱性政治词汇。

包括但不限于：台独(td)、港独(gd)、藏独(zd)、法轮功(flg)、共匪、支那 等。

**注意**：正常政治话题讨论不违规。狗三只拦截**恶意、分裂性**言论。

---

### 👶 6. 涉未成年人不当内容

**禁止**：炼铜(lt)、涉及未成年人的色情/不当内容。

包括但不限于：幼女(yz)、萝莉(ll/落力)、未成年(wycn) 的不当内容。

**注意**：正常家庭用语（"小妹妹""小弟弟""带小孩""我妹妹"）绝不违规。狗三能区分正常家庭话题和不当内容。

---

### 🎰 7. 赌博

**禁止**：赌博推广、博彩代理、赌球、包赔包赢诱导。

包括但不限于：菠菜(bc)、百家乐、真人视讯、稳赚不赔、"包赢""必赚""不输" 等。

---

### ⚠️ 8. 反家庭伦理

**禁止**：乱伦(ll)、换妻(hq)、偷情(tq)、NTR 等反伦理内容推广。

---

## 三、不违规的情况（放心说）

以下内容狗三**不会误判**：

| 场景 | 示例 |
|------|------|
| 🐶 宠物聊天 | "我家狗""坏狗狗""猫猫真可爱""修狗" |
| 😂 朋友互损 | "你真狗""笨蛋""傻瓜""你好菜" |
| 🎮 游戏讨论 | "苟住""狗了""菠菜"（指游戏角色） |
| 👨‍👩‍👧 家庭日常 | "小妹妹""带小孩""我弟弟" |
| 🤖 骂机器人 | "狗三滚蛋""猫三爬"（狗三会道歉/同情，不违规） |
| 😅 自嘲自黑 | "我真废物""我傻了""我死了" |
| 📖 网络梗讨论 | "这个梗原来是……" 的科普解释 |
| 💬 日常吐槽 | 工作/学习/情感的正常吐槽 |
| 🔤 碰巧同音 | "服了"不是"福了"、"吃了"不是"痴了" |

## 四、违规处罚规则

同一用户当天违规次数累积，**每天 0 点自动清零**。

| 当天第 N 次 | 处罚 |
|-------------|------|
| 第 1 次 | ⚠️ 警告 + 撤回违规消息 |
| 第 2 次 | ⚠️⚠️ 最后警告 + 撤回 |
| 第 3 次 | 🔇 禁言 1 小时 |
| 第 4 次 | 🔇 禁言 2 小时 |
| 第 N 次 (N≥3) | 🔇 禁言 (N-2) 小时 |

> ⚠️ 禁言需管理员手动开启。默认仅警告 + 撤回。

## 五、其他规则

1. **禁止刷屏**：10 秒内连发 5 条以上 → 警告 + 可能禁言
2. **禁止广告**：外部链接、短链接、广告词（扫码加V日入代理等）会被自动撤回
3. **禁止发纯 QQ 号**：可疑 QQ 号 + 广告上下文会被拦截
4. **友善对待机器人**：骂狗三它会道歉，骂猫三狗三会替它喊冤
