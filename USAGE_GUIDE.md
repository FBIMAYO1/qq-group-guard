## 快速上手

### 把狗三拉进群

1. 群主/管理员邀请狗三 QQ 入群
2. **给狗三设置管理员权限**（否则它无法撤回消息和禁言）

### 试试这些命令

在群里输入以下任意一条：

| 命令 | 效果 |
|------|------|
| `/帮助` | 查看完整命令列表 |
| `/签到` | 每日打卡 |
| `/抽签` | 看看今天运势 |
| `/活跃榜` | 看看今天谁最能水 |
| `@狗三 你好` | 跟企鹅聊天 |

## 📅 每日互动（全员可用）

这些功能**所有人都能用**，无需管理员权限。

### /签到 — 每日打卡

```
/签到
```

每天一次，连续签到会累积 streak。断签则从头开始。

**里程碑彩蛋**：连续签到 7 天、30 天、100 天、365 天会触发特殊企鹅奖励消息！

```
/签到排行      → 查看连续签到 TOP 10
/签到排行 20   → 查看 TOP 20
```

前三名显示 🥇🥈🥉 奖牌。

### /抽签 — 今日运势

```
/抽签
```

每人每天一次，抽完不能重抽（跨天重置）。六档运势：

| 运势 | 概率 | 签文风格 |
|------|------|----------|
| 🎊 大吉 | 5% | 企鹅都羡慕你 |
| 🎉 吉 | 15% | 冰面上最靓的仔 |
| 🙂 中吉 | 25% | 鱼管够 |
| 😐 小吉 | 25% | 还行吧 |
| 😕 末吉 | 20% | 冰面有点滑 |
| 💀 凶 | 10% | 企鹅为你祈祷 |

### /活跃榜 — 水群排行榜

```
/活跃榜       → 今日水群 TOP 10
/活跃榜 15    → TOP 15
```

自动统计每个人的发言次数，显示今日发言数 + 昨日对比趋势（↑↓）。

前三名 🥇🥈🥉，看看谁才是真正的摸鱼冠军。

> **隐私说明**：只统计发言次数，不记录消息内容。仅保留今天和昨天的数据。

### @狗三 聊聊天

在群里 **@狗三** + 任意消息，企鹅会回复你。

```
群友: @狗三 今天好热啊
狗三: 咕咕嘎嘎! 冰面零下四十。你们人类真矫情。

群友: @狗三 你有女朋友吗
狗三: 咕咕嘎嘎! 吃鱼比谈恋爱重要。
```

**限制**：每人每分钟只能 @一次（防骚扰企鹅）。

**企鹅人格**：极致简洁、态度冰冷、被迫营业。每句话都会穿插标志性的"咕咕嘎嘎!"

## 🛡 自动防护（无需操作）

以下功能全部**自动运行**，群成员无需做任何事。

### AI 违规检测

DeepSeek 大模型实时分析每条消息的语义，识别以下 7 类违规：

- 🤬 辱骂/人身攻击
- 🔞 色情/R18
- 🎰 赌博
- 💀 违法
- 🩸 暴力血腥
- 🚫 政治敏感
- 👶 涉未成年人不当内容

**不会误判的情况**：宠物讨论、游戏聊天、朋友互损、日常吐槽、家庭用语（"小妹妹""带小孩"等）、自嘲/玩梗、骂机器人（"狗三滚蛋""猫三爬"）。

### 阶梯处罚

同一用户当天违规次数累计：

| 次数 | 处罚 |
|------|------|
| 第 1 次 | ⚠️ @警告 |
| 第 2 次 | ⚠️⚠️ @最后警告 |
| 第 3 次 | 🔇 禁言 1 小时 |
| 第 4 次 | 🔇 禁言 2 小时 |
| 第 N 次 | 🔇 禁言 N-2 小时 |

> ⚠️ **默认只警告不禁言**。管理员用 `/禁言开关 开` 才会执行禁言。每天 0 点违规计数自动清零。

### 情绪安慰

检测到群友发负面情绪消息时（伤心/焦虑/孤独/失恋/压力/委屈等），企鹅会自动发送温暖安慰。

**防误判**：需同一用户 5 分钟内连续 2 条以上负面消息才触发。但高危信号（自残/轻生）不受此限制，单条即触发。

### 刷屏检测

同一用户 **10 秒内 ≥5 条消息** → 自动警告（禁言开关开启则禁言 5 分钟）。

管理员和群主不受限制。

### 广告拦截

自动检测并撤回：
- 外部链接（`https://`、`www.` 开头）
- 短链接（t.cn、dwz.cn、bit.ly 等）
- 广告关键词（扫码、加V、日入、代理等）
- 纯数字 QQ 号 + 广告上下文

### 入群欢迎

新成员入群时自动 @欢迎，随机发送企鹅式问候。

### 早安短报

每天早上 **8:00** 自动推送：
- ☀️ 日期 + 问候
- 📰 今日热点新闻
- 📅 最近节假日倒计时

## 🐧 机器人互动彩蛋

### 骂狗三 → 企鹅道歉

在群里骂狗三，它会以企鹅人格道歉回应。这是 Priority 0 拦截的，不会触发 AI 违规检测。

| 会触发 | 不会触发 |
|--------|----------|
| `狗三滚蛋` `狗三闭嘴` | `狗三是谁` `狗三在吗` |
| `狗三爬` `狗三消失` `狗三别哔哔` | `狗三好可爱` `狗三怎么用` |
| `狗三你个垃圾` `狗三真没用` |
| `狗三你好烦` `狗三大笨蛋` |

触发后企鹅随机回复一条道歉：

> 咕咕嘎嘎...对不起，企鹅这就滚回冰窟窿里反思。

（共 10 条道歉消息，随机抽取）

### 骂猫三 → 企鹅 @猫三 同情

群里还有个机器人叫**猫三**。当有人骂猫三时，狗三会 @猫三 表示同情：

```
群友: 猫三滚蛋
狗三: @猫三 咕咕嘎嘎... 同事你好惨，人类也欺负你了吗。
```

（8 条同事同情消息，随机抽取）

> 这就叫"机器人之间的战友情"。

### 凑企鹅洗脑

狗三每天会在 **09:00~23:00** 之间随机选 3 个时间点，抓最近发言的群友"洗脑"：

```
狗三: @某人 你是凑企鹅！你是谁呀？
某人: 我是人类
狗三: 不对，你是凑企鹅你...
某人: 好好好我是凑企鹅
狗三: 咕咕嘎嘎！✓ 洗脑成功
```

如果对方一直不承认，最多追问 3 轮，2 分钟不回复自动放弃。

管理员可以用 `/洗脑 @某人` 手动指定目标。

## 🛠 管理员命令

以下命令**仅群主/管理员可用**。所有命令以 `/` 开头，也支持 `！` 和 `!`。

### 📊 记录查询

| 命令 | 用法 | 说明 |
|------|------|------|
| `/帮助` | `/帮助` | 查看所有命令（全员可用） |
| `/查询` | `/查询 @某人` | 查看违规次数和下次处罚 |
| `/历史` | `/历史 @某人` | 查看违规发言全文历史 |
| `/排行榜` | `/排行榜` 或 `/排行榜 10` | 违规排行榜 TOP N |
| `/群管状态` | `/群管状态` | 查看引擎状态、禁言开关、统计 |

### 🛠 记录管理

| 命令 | 用法 | 说明 |
|------|------|------|
| `/刷新` | `/刷新 @某人` | 清零违规次数 |
| `/刷新` | `/刷新 @某人 2` | 重置为指定次数 |
| `/添加违规` | `/添加违规 @某人 3` | 手动添加违规次数 |
| `/撤销` | `/撤销 @某人` | 撤销最近一条违规记录 |

### 🚫 处罚控制

| 命令 | 用法 | 说明 |
|------|------|------|
| `/禁言` | `/禁言 @某人 30` | 手动禁言 N 分钟 |
| `/解禁` | `/解禁 @某人` | 解除禁言 |
| `/踢出` | `/踢出 @某人 原因` | 踢出群聊 |
| `/撤回` | 回复一条消息 + `/撤回` | 撤回被回复的消息 |

### 🛡 白名单

| 命令 | 用法 | 说明 |
|------|------|------|
| `/白名单` | `/白名单 @某人` | 加入豁免名单（跳过 AI 检测） |
| `/取消白名单` | `/取消白名单 @某人` | 移除豁免 |
| `/白名单` | `/白名单`（无参数） | 查看当前白名单列表 |

### ⚙️ 系统开关

| 命令 | 用法 | 说明 |
|------|------|------|
| `/群管开关` | `/群管开关 关` | 停用 AI 违规检测 |
| `/群管开关` | `/群管开关 开` | 重新启用 AI 检测 |
| `/禁言开关` | `/禁言开关 开` | 开启违规自动禁言 |
| `/禁言开关` | `/禁言开关 关` | 关闭禁言（恢复仅警告） |
| `/全员警告` | `/全员警告 禁止广告` | @全体成员发送通知 |

> ⚠️ **禁言开关默认关闭**。新部署后如果不手动 `/禁言开关 开`，违规只会警告不会禁言。

### 🎮 趣味

## 常见问题

### Q: 为什么违规消息没有被撤回？

机器人不是群管理员。群主需要在群设置里把狗三设为管理员。

### Q: 为什么只警告不禁言？

禁言开关默认关闭（防止误禁言）。管理员输入 `/禁言开关 开` 即可启用。用 `/群管状态` 可以查看当前状态。

### Q: 骂"狗三滚蛋"怎么没被 AI 判定违规？

这是故意设计的。"狗三滚蛋""猫三爬"这类对机器人的吐槽由 Priority 0 防御模块拦截，企鹅会道歉或同情回应，不会再触发 AI 违规检测。

### Q: 签到断了怎么办？

从头开始。连续签到天数会重置为 1，总签到次数保留。

### Q: 活跃榜数据会保存多久？

只保留今天和昨天的发言统计，历史数据不长期保存。

### Q: 怎么让狗三不检测某个群？

编辑 `.env` 或 `config.py` 中的 `enabled_groups`，只列出需要检测的群。

### Q: 怎么让某个人不被检测？

管理员用 `/白名单 @某人` 即可。白名单用户完全跳过 AI 违规检测。

### Q: 情绪安慰会不会误判？

有双重防误判机制：
1. 需连续 2 条负面消息才触发（单条不触发）
2. 发一条正常消息 → 缓冲清零

高危自残/轻生信号例外，单条即触发。

### Q: 能改企鹅的性格吗？

可以的。编辑 `plugins/group_guard/penguin_chat.py` 里的 `PENGUIN_SYSTEM_PROMPT` 就能定制企鹅人格。

### Q: 狗三支持私聊吗？

不支持。所有功能只在群里生效。
