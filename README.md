# 🐶 狗三 QQ群智能管理机器人

基于 **NoneBot2 + NapCat + DeepSeek AI** 的QQ群自动管理机器人。纯AI语义判断，精准识别辱骂、色情、赌博、违法等违规内容并自动处罚。

## ✨ 特性

- 🧠 **纯AI判断** — 基于DeepSeek大模型，无需维护关键词库
- 🔍 **识破变体** — 自动识别谐音字（傻福→傻逼）、形近字（冯→妈）、拼音缩写（nmsl）、拆字
- 📏 **阶梯处罚** — 两次警告 → 第三次起禁言等差递增
- 🗑️ **自动撤回** — 违规消息瞬间撤回
- 🛡 **防误判** — 宠物讨论、外号称呼、朋友互损、游戏聊天统统不误伤
- 📋 **16个管理命令** — 查询/刷新/禁言/踢出/白名单/排行榜等
- 💾 **持久化存储** — 违规记录永久保存，支持查看历史
- 🐧 **凑企鹅角色扮演** — @机器人触发冰企鹅回复，咕咕嘎嘎! 每人每分钟限1次

## 🏗 架构

```
QQ服务器 ←→ NapCat (QQNT协议层)
                ↕ WebSocket
           NoneBot2 (机器人框架)
                ↕
           DeepSeek AI (语义判断)
```

| 层 | 技术 | 作用 |
|----|------|------|
| 协议层 | [NapCat](https://github.com/NapNeko/NapCatQQ) | 对接QQ协议，收发消息 |
| 框架层 | [NoneBot2](https://nonebot.dev/) | 消息路由、插件管理 |
| AI层 | [DeepSeek](https://platform.deepseek.com/) | 语义级违规内容识别 |

## 🚀 快速开始

### 1. 环境要求

- Python 3.10+
- Windows / Linux / macOS
- 一个QQ小号（作为机器人）
- DeepSeek API Key

### 2. 安装依赖

```bash
pip install nonebot2 nonebot-adapter-onebot "nonebot2[websockets]" openai
```

### 3. 配置NapCat（协议层）

1. 从 [NapCat Releases](https://github.com/NapNeko/NapCatQQ/releases) 下载 Windows 版
2. 解压运行 `napcat.bat`，用机器人QQ号扫码登录
3. 访问 `http://127.0.0.1:6099` 进入管理面板
4. 在网络配置中添加 **正向WebSocket服务器**：
   - 地址：`127.0.0.1`
   - 端口：`3001`
5. 保存

### 4. 配置机器人

复制 `.env.example` 为 `.env`，填入你的配置：

```env
SUPERUSERS=["你的QQ号"]
DEEPSEEK_API_KEY=你的DeepSeek_Key
```

> DeepSeek Key 从 https://platform.deepseek.com 免费获取，费用极低（每条消息约0.3分钱）

### 5. 启动

**方式一：一键启动（推荐）**

双击项目根目录下的 `start.bat`，自动启动 NapCat + 机器人。

**方式二：手动启动**

```bash
# 1. 双击运行 NapCat 目录下的 napcat.bat，扫码登录
# 2. 启动机器人
python bot.py
```

看到 `Bot xxxxxxx connected` 即表示连接成功。

### 6. 设置群管理员

将机器人QQ号在目标群里**设为管理员**，否则无法禁言和撤回。

## ⚖ 处罚规则

| 违规次数 | 处罚 |
|---------|------|
| 第1次 | ⚠️ @警告 |
| 第2次 | ⚠️⚠️ @最后警告 |
| 第3次 | 🔇 禁言1小时 |
| 第4次 | 🔇 禁言2小时 |
| 第N次(N≥3) | 🔇 禁言(N-2)小时，等差递增 |

机器人能识别以下类别的违规内容及其**所有变体**：

| 类别 | 包括的变体 |

## 🔧 自定义

### 修改AI判断逻辑

编辑 `plugins/group_guard/ai_checker.py` 中的 `SYSTEM_PROMPT`。

### 修改处罚规则

编辑 `plugins/group_guard/punisher.py` 中的 `calculate_punishment` 方法。

### 添加白名单

在群里发 `/白名单 @某人` 即可，或在 `.env` 的 `whitelist_users` 中添加QQ号。

### 仅在特定群启用

在 `.env` 中添加 `enabled_groups` 或在代码 `config.py` 中配置。

## ⚠ 注意事项

1. 机器人QQ号**必须是群管理员**，否则禁言和撤回无法执行
2. 建议使用**小号**作为机器人，避免主号被风控
3. `.env` 文件包含 API Key，**不要上传到公开仓库**
4. DeepSeek API 按量计费，每条消息约 0.3 分钱，建议先充 20 元
5. 群主和管理员的消息不会被检测（防止误伤自己）
