# 🐧 狗三 — QQ 群 AI 智能管家

基于 **NoneBot2 + NapCat + DeepSeek AI** 的全能型 QQ 群管理机器人。

不只是拦截违规 — 它还**陪你聊天、检测情绪、定时洗脑、推送早报**，是一只活在群里的赛博企鹅。

## 🏗 架构

```
QQ 服务器 ←→ NapCat（QQNT 协议层）
                 ↕ WebSocket
            NoneBot2（机器人框架）
                 ↕
            DeepSeek AI（语义判断 / 聊天 / 情绪检测）
```

| 层 | 技术 | 作用 |
|----|------|------|
| 协议层 | [NapCat](https://github.com/NapNeko/NapCatQQ) | 对接 QQNT 协议，收发消息 |
| 框架层 | [NoneBot2](https://nonebot.dev/) | 消息路由、插件管理、事件调度 |
| AI 层 | [DeepSeek](https://platform.deepseek.com/) | 违规识别、企鹅聊天、情绪检测 |

## 🚀 快速开始

### 1. 环境要求

- Python 3.10+
- Windows / Linux / macOS
- 一个 QQ 小号（作为机器人）
- DeepSeek API Key（[免费获取](https://platform.deepseek.com/)）

### 2. 安装依赖

```bash
pip install nonebot2 nonebot-adapter-onebot "nonebot2[websockets]" openai httpx
```

### 3. 配置 NapCat（协议层）

1. 从 [NapCat Releases](https://github.com/NapNeko/NapCatQQ/releases) 下载
2. 解压运行 `napcat.bat`，用机器人 QQ 号扫码登录
3. 访问 `http://127.0.0.1:6099` → 网络配置 → 添加**正向 WebSocket**：
   - 地址：`127.0.0.1`，端口：`3001`

### 4. 配置机器人

复制 `.env.example` 为 `.env`：

```env
DRIVER=~fastapi+~websockets
HOST=127.0.0.1
PORT=8080
SUPERUSERS=["你的QQ号"]
COMMAND_START=["/", "！", "!"]
ONEBOT_WS_URLS=["ws://127.0.0.1:3001"]
DEEPSEEK_API_KEY=你的DeepSeek_Key
```

### 5. 启动

双击 `start.bat` 一键启动，或在终端：

```bash
python bot.py
```

看到 `Bot xxxxxxx connected` 即连接成功。

### 6. 设置群管理员

将机器人 QQ 在目标群中**设为管理员**，否则无法禁言和撤回。

## 🧠 AI 违规检测

纯 AI 语义判断，不使用关键词匹配。能识别以下 7 类违规及其**所有变体**：

| 类别 | 覆盖范围 |
|------|----------|
| 🤬 **辱骂/诅咒** | 傻福→傻逼、尼冯四了→你妈死了、nmsl/cnm/sb 等拼音缩写、你劝架四了→你全家死了 |
| 🔞 **R18/色情** | 约炮/性暗示/色情资源/擦边球、月抛→约炮、瑟琴→色情、yp/sq 等缩写 |
| 🎰 **赌博** | 菠菜/赌球/百家乐、bc 缩写、稳赚不赔诱导、代理拉人头 |
| 💀 **违法** | 毒品/诈骗/枪支/违禁品、假证代考、dp/zp 缩写 |
| 🩸 **暴力血腥** | 虐待/自残/虐杀/血腥内容 |
| 🚫 **政治敏感** | 分裂主义/邪教/颠覆言论、td/gd/flg 缩写 |
| 👶 **炼铜** | 涉及未成年的不当内容、lt 缩写 |

### 不误判的场景

- 🐶 宠物讨论（"我家狗""坏狗狗""猫猫""修狗"）
- 😂 外号称呼（"狗三""狗哥""小猫""老狗"）
- 🎮 游戏聊天（"苟住""狗了""好菜""这怎么打"）
- 💬 朋友互损（"你真狗""笨蛋""傻瓜"）
- 📖 日常吐槽/情感讨论/学习工作聊天
- 🔤 碰巧同音但非违规（"服了"≠"福了"、单个字巧合）

## ⚖️ 阶梯处罚

每天 0 点自动清零，处罚规则：

| 违规次数 | 处罚 | 说明 |
|----------|------|------|
| 第 1 次 | ⚠️ @警告 | 提醒注意群规 |
| 第 2 次 | ⚠️⚠️ @最后警告 | 告知再犯将禁言 |
| 第 3 次 | 🔇 禁言 1 小时 | — |
| 第 4 次 | 🔇 禁言 2 小时 | — |
| 第 N 次 (N≥3) | 🔇 禁言 N-2 小时 | 等差递增 |

同时自动撤回违规消息（需管理员权限）。

## 🎯 凑企鹅洗脑

每日自动在 09:00~23:00 之间随机生成 3 个触发时间（间隔 ≥ 2 小时），到时揪出最近发言的群成员进行"洗脑"：

| 机制 | 说明 |
|------|------|
| 触发 | 每日 3 次随机时间点，发给该群最近发言的普通成员 |
| 追问 | @TA "你是凑企鹅！你是谁呀？"，最多追问 3 轮 |
| 承认 | 回复含"凑企鹅" → "咕咕嘎嘎！"洗脑成功 |
| 否认 | 追问"不对，你是凑企鹅你…"，直到承认或满 3 轮 |
| 超时 | 2 分钟不回复自动放弃 |
| 不重复 | 当天已被选过的人不会再次被选 |
| 豁免 | 不洗脑管理员/群主/机器人自身 |
| 手动 | `/洗脑 @某人` 管理员可手动触发 |

## 💚 情绪安慰

检测群友的负面情绪发言，以企鹅身份发送温暖安慰。**高危情况（自残/轻生）即时响应。**

### 防误判机制

| 层级 | 机制 |
|------|------|
| **AI 层** | 核心原则"宁可漏过不可误判"；模糊短句/玩梗/日常吐槽/游戏聊天一律不判负面 |
| **缓冲层** | 需同一用户 5 分钟内**连续 ≥2 条**消息都被判负面，才触发安慰 |
| **清零** | 用户发一条非负面消息 → 缓冲立即清零 |
| **高危** | 自残/轻生信号**不受缓冲限制**，单条即触发 |

### 覆盖的情绪

伤心、焦虑、压力、孤独、失恋、委屈、自我否定、愤怒（自身情绪）、亲人宠物离世、高危（自残/轻生）

## 🔧 自定义

### 修改违规检测逻辑

编辑 `plugins/group_guard/ai_checker.py` 中的 `SYSTEM_PROMPT`。

### 修改处罚规则

编辑 `plugins/group_guard/punisher.py` 中的 `calculate_punishment` 方法。

### 修改企鹅人格

编辑 `plugins/group_guard/penguin_chat.py` 中的 `PENGUIN_SYSTEM_PROMPT`。

### 修改情绪检测敏感度

编辑 `plugins/group_guard/comfort.py`，调整 `MIN_NEGATIVE_COUNT`（默认 2）和 `EMOTION_WINDOW`（默认 300 秒）。

### 指定启用的群

在 `.env` 中配置 `enabled_groups` 或在代码 `config.py` 中设置 `plugin_config.enabled_groups`。

## ⚠️ 注意事项

1. 机器人 QQ 号**必须设为群管理员**，否则禁言和撤回无法执行
2. 建议使用**小号**作为机器人，避免主号被风控
3. `.env` 包含 API Key，**不要上传到公开仓库**（已配置 `.gitignore`）
